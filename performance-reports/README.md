# goxpy performance reports

This folder (will) contains independent reports on the timing performance of goepxyriment.

Use a [pull request](https://github.com/chrplr/goxpyriment/pulls) or send your `report.md` file (markdown format; zipped if you need to include additionnale files, e.g., images) to <christophe@pallier.org>

Please do not forget to include a precise description of the hardware used, as provided, e.g, by [go-inxi](http://chrplr/github.io/go-inxi).
